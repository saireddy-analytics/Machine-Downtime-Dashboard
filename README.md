# Machine Downtime Dashboard

## Overview
This project analyzes machine downtime across multiple factories using Tableau. The goal is to identify which factories and machine types experience the highest failures.

## What I Did
- Cleaned and prepared the dataset
- Created calculated fields to measure downtime
- Built interactive dashboards using Tableau
- Compared downtime across factories and device types

## Key Insights
- Identified the factory with the highest downtime
- Found the most frequently failing machine types (e.g., LaserWelder, LaserCutter)
- Enabled quick comparison using visual dashboards

## Tools Used
- Tableau
- Excel
